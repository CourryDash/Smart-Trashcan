# Smart-Trashcan
Project Agile [Kelompok O7] - due 31 Maret 2026

Fitur-fitur : 
1. **Waste Sorting System** : Menyortir sampah menjadi organik, anorganik, dan B3

2. **Full Capacity Notification** : Memberikan notifikasi saat tempat sampah penuh

3. **Security Alarm System (Sirine & Kejut Listrik - Simulasi)** : Mengaktifkan alarm saat terjadi akses tidak sah

4. **Petugas Login System** : Petugas dapat login untuk mengosongkan sampah

5. **Login Error Handling** : Menampilkan pesan error saat login gagal

6. **Activity Log System** : Mencatat aktivitas seperti login dan pengosongan sampah

7. **Trashcan Capacity Percentage** : Menampilkan persentase kepenuhan tempat sampah

8. **Automatic Capacity Increase** : Kapasitas bertambah setiap ada sampah masuk

9. **Trashcan Reset System** : Petugas dapat mengosongkan tempat sampah

10. **Statistics Monitoring System** : Menampilkan data kapasitas dan log aktivitas

11. **Admin Full Access Login** : Admin dapat mengakses seluruh sistem dan melihat statistik

12. **Access Validation System** : Memastikan hanya ID tertentu yang dapat login

13. **User Interface (UI)** : Antarmuka berbasis terminal untuk interaksi pengguna

14. **Role-Based Access Control** : Membatasi akses berdasarkan peran (User, Petugas, Admin)

15. **Interactive Menu System** : Menu interaktif untuk menjalankan fitur sistem
